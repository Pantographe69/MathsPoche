# Dérivation — calcul de la dérivée

## Définition

La **dérivée** d'une fonction $f$ en un point $a$ est la limite du taux de variation :

$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

Géométriquement, $f'(a)$ est le **coefficient directeur** de la tangente à la courbe au point d'abscisse $a$.

---

## Dérivées usuelles

| Fonction | Dérivée |
|----------|---------|
| $f(x) = k$ (constante) | $f'(x) = 0$ |
| $f(x) = x^n$ | $f'(x) = nx^{n-1}$ |
| $f(x) = \sqrt{x}$ | $f'(x) = \dfrac{1}{2\sqrt{x}}$ |
| $f(x) = \dfrac{1}{x}$ | $f'(x) = -\dfrac{1}{x^2}$ |
| $f(x) = e^x$ | $f'(x) = e^x$ |
| $f(x) = \ln x$ | $f'(x) = \dfrac{1}{x}$ |

---

## Règles de calcul

### Linéarité
$$(f + g)' = f' + g' \qquad (kf)' = kf'$$

### Produit
$$(fg)' = f'g + fg'$$

**Exemple :** $h(x) = x^2 \cdot e^x$

$$h'(x) = 2x \cdot e^x + x^2 \cdot e^x = e^x(2x + x^2) = xe^x(x+2)$$

### Quotient
$$\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$$

### Composée
$$(f \circ g)'(x) = g'(x) \cdot f'(g(x))$$

**Exemple :** $h(x) = (3x+1)^4$

$$h'(x) = 4(3x+1)^3 \cdot 3 = 12(3x+1)^3$$

---

## Dérivée et variations

| Signe de $f'$ | Variation de $f$ |
|---------------|------------------|
| $f'(x) > 0$ sur $I$ | $f$ **croissante** sur $I$ |
| $f'(x) < 0$ sur $I$ | $f$ **décroissante** sur $I$ |
| $f'(a) = 0$ | Possible extremum en $a$ |

---

## Tableau de variations — méthode

**Étudier les variations de $f(x) = x^3 - 3x + 1$**

1. $f'(x) = 3x^2 - 3 = 3(x^2 - 1) = 3(x-1)(x+1)$
2. $f'(x) = 0 \Rightarrow x = -1$ ou $x = 1$
3. Signe de $f'$ :

| | $-\infty$ | | $-1$ | | $1$ | | $+\infty$ |
|--|--|--|--|--|--|--|--|
| $f'(x)$ | | $+$ | $0$ | $-$ | $0$ | $+$ | |
| $f$ | ↗ | | max | ↘ | | min | ↗ |

---

## Équation de la tangente

La tangente à $\mathcal{C}_f$ au point d'abscisse $a$ a pour équation :

$$y = f'(a)(x - a) + f(a)$$

**Exemple :** Tangente à $f(x) = x^2$ en $x = 2$

$$f(2) = 4, \quad f'(2) = 4 \Rightarrow y = 4(x-2) + 4 = 4x - 4$$
